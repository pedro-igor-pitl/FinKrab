# FinKrab

**FinKrab** é um aplicativo de controle financeiro pessoal desenvolvido com **Spring Boot** no back-end e **React** no front-end. Ele permite que os usuários gerenciem contas, transações e categorias de forma simples e intuitiva.

---

## Funcionalidades

- Criar e gerenciar contas bancárias ou carteiras.
- Registrar transações (Receitas e Despesas).
- Categorizar transações com cores para melhor visualização.
- Consultar saldos e movimentações.

---

## Tecnologias

- **Back-end:** Spring Boot, JPA, PostGreSQL
- **Front-end:** React, Axios, CSS
- **Versionamento:** Git / GitHub
- **Conteinerização:** Docker

---

## Estrutura do Projeto

/finkrab

---

## Como rodar o projeto

### Back-end

1. Configure um banco de dados PostGreSQL.
2. Atualize o `application.properties` com credenciais do banco.
3. Execute `mvn spring-boot:run`.

### Front-end

1. Entre na pasta `/frontend`.
2. Execute `npm install` para instalar dependências.
3. Execute `npm start` para rodar a aplicação.
